- The grader will try to find the first link (Markdown style) in the issue body and then try to grade it. You can let GitHub upload your code by just dragging the packed file (We recommend you rename it so its suffix is .txt) into the input box when creating issues. A legal link should have the following format:

```
[Any text](The link to your code package file)
```

- DO NOT post any image link or other link before your code package link, or they may not be judged.

- DO NOT upload code without encryption. We will consider a penalty if you don't notice us to delete it in time.

- DO NOT try to hack the grader or do other cheating behaviors. There's going to be serious results.
